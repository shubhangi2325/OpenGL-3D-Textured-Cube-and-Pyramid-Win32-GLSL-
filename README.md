# OpenGL Win32 Textured Cube and Pyramid Renderer

This project demonstrates rendering of 3D textured shapes using the OpenGL programmable pipeline in C. The application creates a Win32 window and initializes the OpenGL rendering context manually without using external libraries such as GLFW or GLUT.

The program renders a rotating cube and pyramid with texture mapping using GLSL shaders.

## Features

• Rendering of 3D Cube and Pyramid  
• Texture mapping using OpenGL  
• Vertex and Fragment shaders written in GLSL  
• Model, View, Projection matrix transformations  
• GPU buffer management using VAO and VBO  
• Win32 API based window and OpenGL context creation  
• Real-time 3D rendering
